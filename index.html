cooking_auth

A user authentication microservice for an online cooking forum. Check out the project's documentation.

Prerequisites

  • Docker

Initialize the project

Start the dev server for local development:

docker-compose up

Create a superuser to login to the admin:

docker-compose run --rm web ./manage.py createsuperuser

Swagger-ui

You can find the swagger ui at http://localhost:7070/api/v1/schema/swagger-ui
